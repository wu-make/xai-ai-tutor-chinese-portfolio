## Portfolio Media

The main media files are provided in the GitHub Release section:

### 1. Real-Time Mandarin ASR + TTS Demo

**File:** `demo_01_asr_tts_robotic_scrub_nurse.mp4`

This video demonstrates a Mandarin speech-interaction workflow for a robotic scrub nurse scenario. The system captures a physician-style spoken command, performs speech recognition, converts the recognized utterance into a task-oriented command, and generates a spoken response through a TTS module.

This demo shows experience with:

* Mandarin speech recognition
* Spoken-command interpretation
* STT/TTS system integration
* Voice-based human-robot interaction
* Medical-domain Chinese command processing

### 2. Voice-Controlled Endoscopic Robot Demo

**File:** `demo_02_voice_control_endoscopic_robot.mp4`

This video demonstrates an endoscopic robot control workflow based on real-time Mandarin voice commands. The system recognizes spoken directional commands such as “向左” and “向右”, converts them into structured control instructions, and executes the corresponding robot motion.

This demo shows experience with:

* Real-time Mandarin ASR
* Voice-command parsing
* Large speech model usage
* Speech-to-action robotic control
* Human-robot interaction in medical robotics

### 3. Self-Recorded Mandarin Audio Sample

**File:** `audio_01_general_command_cup.mp3`

This is a short self-recorded Mandarin audio sample containing the utterance:

> 请给我水杯。

The sample is used to demonstrate simple Mandarin audio transcription and structured annotation. It is not collected from a real user study, hospital environment, or private conversation.

### 4. JSON Audio Annotation Example

**File:** `audio_01_general_command_cup_annotation.json`

This file provides a JSON-format annotation for the self-recorded audio sample. It includes:

* Audio ID
* File name
* Language tag
* Timestamp boundary
* Transcript
* Normalized transcript
* Pinyin
* Intent label
* Entity label
* Speech-quality metadata
* Review status
* Privacy status

The annotation demonstrates my understanding of structured audio data preparation for ASR and audio-classification model training.
